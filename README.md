# 🔋 Geely EX2 - Calculadora de Bateria (PWA)

## Como publicar de GRAÇA e compartilhar com seus amigos

### Passo 1: Criar conta no GitHub (se não tiver)
1. Acesse **github.com**
2. Clique em **Sign Up** e crie sua conta

### Passo 2: Criar o repositório
1. Clique no **+** no canto superior direito → **New repository**
2. Nome do repositório: **geely-ex2-bateria**
3. Marque **Public** (precisa ser público)
4. Clique em **Create repository**

### Passo 3: Enviar os arquivos
1. Na página do repositório, clique em **uploading an existing file**
2. Arraste TODOS os 5 arquivos desta pasta:
   - `index.html`
   - `manifest.json`
   - `sw.js`
   - `icon-192.png`
   - `icon-512.png`
3. Clique em **Commit changes**

### Passo 4: Ativar GitHub Pages
1. Vá em **Settings** (engrenagem) do repositório
2. No menu lateral, clique em **Pages**
3. Em "Source", selecione **Deploy from a branch**
4. Escolha a branch **main** e pasta **/ (root)**
5. Clique em **Save**
6. Aguarde 1-2 minutos

### Passo 5: Acessar e compartilhar!
Seu app estará no link:
```
https://SEU-USUARIO.github.io/geely-ex2-bateria/
```
(troque SEU-USUARIO pelo seu nome de usuário do GitHub)

### Como seus amigos instalam:
1. Abra o link no celular
2. **Android**: aparece banner "Instalar App" automaticamente
3. **iPhone**: Safari → ícone Compartilhar (□↑) → "Adicionar à Tela de Início"
4. Pronto! Vira um ícone como app nativo, funciona offline!

### Botão Compartilhar
O app tem um botão "📤 Compartilhar App" que usa o compartilhamento nativo do celular — 
seus amigos podem reenviar direto pelo WhatsApp, Telegram, etc.

---
**Arquivos incluídos:**
- `index.html` — O app completo
- `manifest.json` — Configuração PWA (nome, ícones, cores)
- `sw.js` — Service Worker (funciona offline)
- `icon-192.png` — Ícone 192x192
- `icon-512.png` — Ícone 512x512
- `README.md` — Este arquivo
